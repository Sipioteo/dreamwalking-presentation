# DreamWalking — Presentazione interattiva

Presentazione zoom-based del progetto di tesi **DreamWalking**: decodifica di
oggetti *immaginati* dall'EEG in oggetti 3D interattivi, attraverso un loop di
co-immaginazione uomo-macchina.

**Sito:** https://sipioteo.github.io/dreamwalking-presentation/

Si parte dalla mappa della pipeline e si entra sistema per sistema. Comandi:
`→` / `Spazio` / click per avanzare, `←` per tornare, `O` per la mappa,
`1`–`9` per saltare a un blocco, `F` per lo schermo intero.
